# 🛒 MERN Stack E-Commerce Application
 A modern full-stack eCommerce platform built with the MERN stack, designed for smooth shopping experience, secure authentication, and scalable architecture.

## 🚀 Project Overview

A fully functional eCommerce web application where users can explore products, manage their cart, save addresses, and place orders seamlessly.  
Built with a scalable MERN architecture to simulate real-world production applications.

## ⚙️ Tech Stack

### Frontend
Built using modern UI technologies for fast and responsive user experience.  
React + Vite + Tailwind CSS ensure speed and clean design.

### Backend
Node.js and Express power the backend APIs with secure authentication and business logic.  
RESTful APIs handle all client-server communication.

### Database
MongoDB Atlas provides flexible and scalable NoSQL data storage.  
Mongoose is used for schema modeling and database operations.


## 👤 User Features

- Browse products with search & filters  
- View detailed product information  
- Add/remove items from cart  
- Save delivery address  
- Place orders securely  


## 🛠️ Admin Features

- Add new products  
- Edit existing products  
- Delete products  
- Manage inventory efficiently  


## 🔐 Authentication System

Secure login and registration system using JWT authentication.  
Passwords are encrypted using bcrypt for maximum security.


## 🌍 Deployment

- Frontend deployed on Netlify for fast global delivery  
- Backend deployed on Render for scalable API hosting  
- Database hosted on MongoDB Atlas cloud cluster  

## 🎯 Project Goal

This project demonstrates full-stack development skills including frontend design, backend API creation, database integration, authentication, and deployment.  
It replicates real-world eCommerce system architecture.
